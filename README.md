# CursoWebMarlon
Curso de ASP.NET Core do para o Marlon Cesar.

## Arquitetura

Este projeto foi pensado para a arquitetura em quatro camadas, sendo: Aplicação, Domínio, Infra-estrutura e Web.
Na parte da Infra, estão as comunicações com banco de dados, no caso, o SQL Server e com as API's externas.

Este projeto é uma API em ASP.Net Core 3.1, que utiliza a arquitetura REST em sua composição.
